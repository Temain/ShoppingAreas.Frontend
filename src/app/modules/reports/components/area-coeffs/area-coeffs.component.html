Коэффициент установочной площади:

Ку = Sоб/Sтз

Sоб - площадь занятая оборудованием, Sтз - площадь торгового зала

Ку = {{ report.equipmentArea }} кв.м. / {{ report.totalArea }} кв.м. = {{ report.coefInstall | number: '1.2-2' }}

Норматив коэффициента - 0.3

Необходимо дополнителльно установить оборудование:

Sтз * 0.3 - Sоб = {{ report.totalArea * 0.3 - report.equipmentArea | number: '1.2-2' }} кв.м.

Коэффициент демонстационной площади:

Кд = Sвт/Sтз

Sвт - площадь выкладки товаров, Sтз - площадь торгового зала

Кд = {{ report.productArea }} кв.м. / {{ report.totalArea }} кв.м. = {{ report.coefDemo | number: '1.2-2' }}

Норматив коэффициента - 0.75

Необходимо увеличить выкладку товаров:

Sтз * 0.75 - Sвт = {{ report.totalArea * 0.75 - report.productArea | number: '1.2-2' }} кв.м.
